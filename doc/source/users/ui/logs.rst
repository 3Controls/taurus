===========================================
Taurus Remote Log Monitor User's Interface
===========================================

.. contents::

Taurus provides a user interface that allows to visualize logs generated by a
given Taurus GUI.


Taurus Remote Log Monitor Application
--------------------------------------

taurusremotelogmonitor allows to visualize Taurus logs for the following
log levels: trace, debug, info, warning, error and critical.
This application can be launched using the following command::

    taurusremotelogmonitor

Run the following command for more details::

    taurusremotelogmonitor --help
